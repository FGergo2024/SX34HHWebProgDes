SZÓKIRAKÓJÁTÉK

Adott egy 4*9-es négyzetháló. A program megad egy - a négyzetháló feletti képhez
kapcsolódó szót vagy mondatot, majd betűit összekeverve elhelyezi ebben a
négyzethálóban.
A játékos feladata az, hogy az összekevert betűkből kirakják a szót
vagy mondatot a kijelölt helyen. A betűket vízszintes vagy függőleges irányban egyenként
lehet mozgatni.

Ha a betű útja elállja egy másik betű, akkor csak a másik betűig tolódik el.
Miután sikerül a betűket a megfelelő helyre mozgatni, a program ezt jelzi, ha viszont valamely
betűket felcseréltük, akkor azokra színkiemeléssel hívja fel a
figyelmünket.

Derékszögű háromszögben igaz Pitagorasz tétele:
a2 + b2 = c2
